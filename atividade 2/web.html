O HTML é a base da web

O HTML é a base da web, e ele nos permite criar páginas com muitos recursos interessantes!

Um dos elementos mais importantes é o texto formatado, como o uso de sublinhado para dar ênfase.

Além disso, podemos destacar partes importantes do conteúdo. Por exemplo, em uma fórmula científica, temos E = mc², onde a energia é igual à massa vezes a velocidade da luz ao quadrado.

Em uma fórmula química, temos H2O, que representa a água.

Às vezes coisas simples como mostrar uma tag HTML como essa <p> podem ser complicadas ao escrever o código.
